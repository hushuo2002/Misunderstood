# 产品需求文档

## 一句话

`星际误解局 (Misunderstood: Earth to Moon)` 是一款地月文化误解卡牌游戏。玩家通过判断场景含义，理解不同生存环境如何塑造不同的礼仪、沟通、饮食、运动和治理方式。

## 核心体验

玩家不只是“答题”，而是在做一次文化翻译：

1. 看到一个容易误解的行为。
2. 从多个解释中选择最合理的一项。
3. 立即得到对错反馈。
4. 阅读标准解释和更深层的环境原因。
5. 在地球组与月球组的回合交替中看到误解是双向的。

最终想留下的感受是：

> 对方不是奇怪，只是活在另一种世界里。

## MVP 范围

已实现或应保留：

- 首页
- 卡组选择
- 世界观导入
- 阵营选择
- 胜利分数选择
- 单人对局
- AI 对手
- 答题反馈
- 分数面板
- 结算页
- 多人房间入口占位
- 中英文 i18n

暂不进入 MVP：

- 真正多人同步
- 账号系统
- 服务器存档
- 卡牌编辑后台
- 支付、排行榜、社交系统

## 用户

- 黑客松评委和现场观众
- 想快速体验设定的轻量玩家
- 对科幻、世界观、文化差异感兴趣的玩家
- 后续内容创作者和卡牌设计者

## 成功标准

- 3 分钟内能讲清设定和玩法。
- 5 分钟内能完成一局或接近一局。
- 每张卡都能提供“误解点”和“环境原因”。
- 新增卡组和卡牌不需要修改核心游戏逻辑。
- 图片生成可以按卡牌增量运行。
- 单机体验稳定，不依赖服务端。

## 设计原则

- 单人优先，多人后置。
- 解释比对错更重要。
- 内容要围绕真实环境约束展开。
- 视觉保持“太空档案终端”方向。
- 游戏逻辑、内容数据、动画表现分层。
- 任何生成或 API 调用都不能暴露密钥。

## 未来产品方向

### 内容扩展

- 新卡组：医疗、教育、家庭、法律、娱乐、灾害应急。
- 卡牌标签：资源、风险、亲密、权力、时间、身体、情绪。
- 难度曲线：从直觉误解到制度误解。

### 创作工具

- 本地卡牌校验脚本。
- 卡牌图片批量生成脚本。
- Prompt 模板和风格锁定。
- 卡牌预览页。

### 多人模式

- 房间码进入。
- 地球组/月球组多人协作。
- 轮流答题或组内投票。
- WebSocket/PartyKit/Supabase Realtime 等实时同步方案。
