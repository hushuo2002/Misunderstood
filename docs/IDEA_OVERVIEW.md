# 项目概览

## 这是什么

`星际误解局 (Misunderstood: Earth to Moon)` 是一个地月文化误解主题的卡牌 Web 游戏。

玩家在地球与月球之间进行文化翻译：一个动作、一顿饭、一个仪式或一条规则，在另一个环境里可能完全不是原本以为的意思。

## 为什么成立

地球和月球不是两个地点，而是两套生存条件：

- 重力不同。
- 光照节律不同。
- 空间密度不同。
- 水和空气的价值不同。
- 风险管理方式不同。
- 社会规则的默认前提不同。

这些条件会影响语言、身体动作、饮食、礼仪、工作和权力结构。游戏把这些看不见的默认前提变成可以体验的误解。

## 当前体验

当前 MVP 是单人优先：

1. 玩家进入首页。
2. 选择卡组。
3. 阅读世界观导入。
4. 选择阵营和目标分数。
5. 与 AI 对手轮流答题。
6. 通过解释理解误解来源。
7. 进入结算。

## 核心设计

重点不是“答对”，而是“理解为什么会答错”。

每张卡应该回答三个问题：

- 表面发生了什么？
- 玩家为什么可能误解？
- 环境条件为什么让正确解释成立？

## 美术方向

继续保留“space archive terminal”：

- 太空档案局
- 复古未来主义
- 干净的科幻终端
- 卡牌像被归档的文化案例
- 动画像资料从牌库中被调取

## 后续重点

短期不是换框架，而是让项目变得更容易扩展：

- 统一卡牌 schema。
- 统一游戏状态模型。
- 把动画从 UI 组件里抽离。
- 建立图片生成流水线。
- 给新增卡牌提供校验和生成脚本。
